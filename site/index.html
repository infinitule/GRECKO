⚠ Scope boundary

Simulation, research, and C2-software only. No fire-control, no RF/jamming/waveform design, no effector or weapon hardware control. Effectors are parameter sets (cost, kill probability); comms denial is an abstract link-probability model. A CI gate (python -m tools.verify_invariants) enforces the boundary.

The headline — PX, Monte Carlo cost-exchange

A magazine-conscious economic allocator intercepts swarm raids at materially lower cost-per-kill than a greedy baseline — and the advantage is largest against the attacks an adversarial league discovered.

Mutual co-evolution — PM, Blue adapts back

The league evolves Red attacks; then Blue adapts its effector loadout and rationing knob to those discovered attacks. The finding: against cheap quadrotor swarms, interceptor capability is not the binding constraint — intercept rate is interceptor-count-limited — so cost is the lever. The rational Blue fields cheap collision drones and slashes cost-per-intercept, and Red cannot claw the cost axis back.

Architecture — entity-component, deterministic, POSG-correct

World state is plain data; systems are pure functions. A fixed 50 Hz timestep plus a seeded RNG make every run reproducible — the SHA-256 of the event log is the replay acceptance criterion. Nothing downstream of sensing may read ground truth.

Hardening — invariants as a gate

Four load-bearing properties are checked mechanically by python -m tools.verify_invariants, which exits non-zero on any violation and runs in CI. The gate is proven to detect violations via negative-control tests, not merely to return green.

POSG static

No production fusion module reads the truth sidecar.

Interlock AST

Exactly one world.assign() in the bridge, guarded by can_engage().

Scope regex

No fire-control / RF-waveform / weapon vocabulary outside disclaimers.

Determinism replay

A fixed scenario replays to a byte-identical log hash.
